# Sophy Chatbot

Sophy is a philosophical chatbot, that is, prepared to hold philosophical conversations of up to 20 topics:
- Love
- Atheism
- Good
- Science
- Ethics
- Happiness
- War
- Justice
- Metaphysics
- Death
- Hate
- Politics
- Reason
- Religion
- Sadness
- Truth
- Life

This chatbot is interfaced with a telegram bot named @s0phy_bot.
